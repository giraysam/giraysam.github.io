Boomerang Cache

Boomerang Cache is a client side caching library based on localStorage and sessionStorage.

Download: Source, Minified
Follow the project on Github project page or report a bug
View Full Documentation
Keywords: Javascript, localStorage, sessionStorage 

                        
// Usage
var boomerang = BoomerangCache.create("bucket_1");
var seconds = 3;
boomerang.set('key', 'Hello World', seconds);
boomerang.get('key', 'Hello World');

Viiny Dragger

Viiny Dragger is a drag and drop plugin for javascript.

Download: Source, Minified
Follow the project on Github project page or report a bug
View Full Documentation
Keywords: Javascript, Drag and Drop 

                        
// Dragging
var obj = document.getElementById('drag-1');
viiny.dragger(obj);

// Snapping
var obj = document.getElementById('drag-1');
viiny.dragger(obj, {
    snapX: 50,
    snapY: 50
});

// it has Axis, Callbacks, Restriction features...
