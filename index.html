Boomerang Cache

BoomerangCache is a client side caching library based on localStorage and sessionStorage.

Download: Source, Minified
Follow the project on Github project page or report a bug
View Full Documentation
Keywords: Javascript, localStorage, sessionStorage 

                        
// Usage

var boomerang = BoomerangCache.create("bucket_1");
var seconds = 3;
boomerang.set('key', 'Hello World', seconds);
boomerang.get('key', 'Hello World');
